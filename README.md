# How To Use

1. Install [espnet](https://espnet.github.io/espnet/installation.html) and `requiremets.txt`.

2. Unzip `data.zip`

```
unzip data.zip
```

3. Prepare data

```
python dataPreparation.py
```

4. Run `s2t.py`

```
python s2t.py
```
